This is a step by step guide on how to create a basic website

Step 1

1.Create a github account.
2.Create a github repository. Name the repository as "your_username.github.io"
I would prefer to use Github desktop to maintain my website as it is more convient
IMPORTANT: One github account can only host one website.

Step 2

1.Learn the basic of HTML(HyperTextMarkupLanguage)
Things to note:every HTML file has
1.Header - Title
2.Body - Navigation tool, Content
3.Footer - Contact infomation, Copyright

Step 3

1.Learn the basics of CSS(Cascading Style Sheet)
2.Plan the layout of your website eg.Title,Navigation tool,Content
3.Plan the details of your website eg. Colors, Fontsize
