3D Printing Laser Cutting TinkerCAD
Final project

Step 1

1) Design the phone stand using Fusion360

2) Export design at STL file

Step 2

1) Import STL file into UltiMaker Cura

2) Slice the model using the recommended setting, use support and adhesion if your model needs it

Recommended settings can be found on FLMS on the machine you are about to book

1) According to FLMS, the recommended layer height is 0.2mm and 5-10% infill

2) After slicing, the estimated time to 3D print is 2hr 35mins

After printing, we need to remove the supports and clean up by filling down the protruding filament
