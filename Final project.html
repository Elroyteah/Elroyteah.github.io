Final project

Ideation

I had brainstormed 3 ideas

Idea 1) Lamp that sense person with ultrasonic sensor and output light. Buttons can be pressed to change the color of the light

Idea 2) Fire/Smoke detector that is shaped like a spider. When it senses fire/smoke, it outputs a buzzer and it starts crawling around

Idea 3) Doorlock that uses a motor to lock and unlock it.

I chose the doorlock as my final project idea as it is more practical in day to day life. I also change the application of the lock into a sercurity box.

1) Matrix keypad to control the motor which is acting as the lock

2) RGB led to control the state of the lock, eg. green for unlock, red for lock

3) LDR to control the leds in the box

TinkerCAD design (draft)

Functions of keypad: press '#' to confirm password input, press '*' to clear password input

Correct password: Press '1234' followed by '#' to turn motor and led turns green

Reset button: Press 'B' followed by '#' to reset motor and led turns blue

Wrong password: Press any combination of buttons followed by '#' and the motor would not move and led turn red

Clear input: If wrong button combination is inserted, press '*' to clear the input numbers and re-enter the correct password

LDR system: When there is light, ie. the user open the box, the LED will light up to show content
